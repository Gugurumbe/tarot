Description détaillée

Représente les cartes face visible sur la table.

Il y a une relation de composition entre un Tapis et des Carte.

Pour utiliser directement un Message, il est nécessaire de lier le protocole.

Voir également
Carte
Message La classe Tapis permet de connaître l'état du tour de jeu, c'est-à-dire les cartes jouées ce tour-ci et les numéros des joueurs (0 à 4) qui jouent ces cartes.

La méthode Tapis::terminer(unsigned int, unsigned int, bool) retourne la liste des cartes gagnées par l' « attaque », c'est-à-dire le joueur qui a pris et le joueur appelé, lorsque le pli est Tapis::complet() const.

Termine le tour. Lorsque le tour est terminé, il est nécessaire d'attribuer les cartes gagnées par l'attaque. Pour cela, il faut connaître le JOUEUR appelé. Cette méthode n'est à appeler que du côté serveur.
